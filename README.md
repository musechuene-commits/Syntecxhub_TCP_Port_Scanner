# 🔐 Syntecxhub TCP Port Scanner

## 📌 Cybersecurity Internship Project

A Python-based **multi-threaded TCP Port Scanner** built for cybersecurity learning and internship submission.  
It scans a target system for open and closed ports using socket programming and concurrency.

---

## 🚀 Features
- Scan single host or domain
- Custom port range scanning
- Multi-threaded scanning for speed
- Real-time results display
- Logs scan results to file
- Error handling and timeout support

---

## 🛠️ Technologies Used
- Python 3
- Socket Programming
- ThreadPoolExecutor (Concurrency)
- Command Line Interface (CLI)

---

## ⚙️ How to Run

```bash
python src/scanner.py --host 127.0.0.1 --ports 1-100
